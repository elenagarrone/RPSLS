How to use:
-----------

Clone the directory:
```shell
$ git clone https://github.com/elenagarrone/RPSLS.git
```
Change into it:
```shell
$ cd RPSLS
```
Install the gems:
```shell
$ bundle install
```
Try it on localhost:
```shell
$ rackup
#then on the browser go to:
localhost:9292
```

How to test it:
--------------
From inside the directory run:
```shell
$ rspec
```
and:
```shell
$ cucumber
```
